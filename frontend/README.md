# Frontend

This project was bootstrapped with [Vite](https://vitejs.dev/) and React.

## Development server

Run `npm run dev` for a dev server. Navigate to `http://localhost:5173/`. The app will automatically reload if you change any of the source files.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Running end-to-end tests

Run `npx cypress run` to execute the end-to-end tests. For interactive mode, use `npx cypress open --e2e`.
